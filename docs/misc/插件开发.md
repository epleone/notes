**位运算**
- 支持位运算
- 添加测试

| 运算符  | 含义        | 示例            |
| ---- | --------- | ------------- |
| `&`  | 按位与       | `5 & 3 = 1`   |
| \|   | 按位或       | `5 \| 3 = 7`  |
| `^`  | 按位异或      | `5 ^ 3 = 6`   |
| `~`  | 按位取反      | `~5 = -6`     |
| `<<` | 左移        | `5 << 1 = 10` |
| `>>` | 算术右移（带符号） | `5 >> 1 = 2`  |
