### 功能

- 大数的支持验证，换底层计算库( `decimal.js`  &  `ml-matrix.js`)
-  在不改变当前布局，且修改最小化的情况下，输入太长时支持换行？
- 自适应主题，通过`utools.isDarkColors`判断
- 搜索推送，`utools.onMainPush`
- 补全排序按频次统计
- [bug] 补全框会被遮挡（第一行不会，第二行后都会被遮挡）
- 支持日期操作的补全
- 更新字符串其它常见操作
- 页面为空时，清空按键禁用
- 添加一个清空动画，清空触发后，历史按钮明显放大，抖动，变色


**高阶操作**
- 支持自定义常数
- 支持自定义函数，支持lambda 函数 ` x, y => x + y`，原生js 函数.  类型咋弄？
- 使用utools数据保存自定义的代码
-  支持调用内部自定义函数 (需要避免循环调用)
-  测试页面

**高阶计算**
- 支持数组/向量 `[ ... ]`
- 支持矩阵 `{ ... }`
-  常用计算
-  可视化矩阵计算的结果，新设计提示图标。点击图标弹出结果显示页面。
-  添加测试


**支持弹窗 + 网页**
-   仿照 `简易计算器`，直接在搜索框得到结果 （不支持变量）。

**日期**

`@` 将日期转时间戳, 添加info
`#` 获得时间间隔， `#1y1m3w4d5h6mm7s8ms`  --> 转成函数，返回`datestamp`对象 ,  添加info

``` js
// 加减法设计
`@... + @...` --> Error

`@... - @...`  --> #   显示毫秒数， 添加info显示 x天x小时x分x秒x毫秒

`@... ± #...`  --> @   , 显示日期，类型是字符串

`@... ± #... > @`  ， 变成时间戳，添加info显示单位ms

`@... > #`   ,  ERROR

`#... > #`  格式化显示， x天x小时x分x秒x毫秒
`#... > #d` 格式化显示， x天
`#... > #h` 格式化显示， x小时
...

`> latex`  对矩阵有用

```

支持 `@now` `@today`
