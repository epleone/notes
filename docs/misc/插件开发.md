### 功能

- 大数的支持验证，换底层计算库
- 自适应主题，通过`utools.isDarkColors`判断
- 搜索推送，`utools.onMainPush`
-  补全排序按频次统计
- 字符串其它常见操作
-  $\pi$ 的多支持
- 修改历史快照，`添加当前页面`,  按钮状态设置为不可用。
- 添加最近删除，每次清空的记录自动保存到最近删除，保留多少条？界面如何设计？
- 添加一个清空动画，清空触发后，历史按钮明显放大，抖动，变色。
- 将历史快照页面分成两部分，点击标题栏切换。


**高阶操作**
- 支持自定义常数
- 支持自定义函数，支持lambda 函数 ` x, y => x + y`，原生js 函数.  类型咋弄？
- 使用utools数据保存自定义的代码
-  支持调用内部自定义函数 (需要避免循环调用)
-  测试页面

**高阶计算**
- 支持数组/向量 `[ ... ]`
- 支持矩阵 `{ ... }`
-  常用计算
-  悬浮显示，提示图标为 `info`
-  添加测试


**支持弹窗 + 网页**
-   仿照 `简易计算器`，直接在搜索框得到结果 （不支持变量）。

**日期**
需要好好设计一下功能
`#2100y`  `#3mo`  `#3w`  `#3d`   `#3h`  `#3m`  `#3s`   `#3ms`  默认单位ms
`#21001021 表示日期`, 返回距离起始时间的时间戳,  `info : y m d h m ms 对应默认的 @ `
`#17:34:53 - #18:15:43  @ =1h19m`  表示时间
`#17:34:53 - #18:15:43  @m = 79m`

`${year}-${month}-${day} ${hours}:${minutes}:${seconds}`

`'YYYY-MM-DD HH:mm:ss'`
