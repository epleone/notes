

# 股票心得

## 如何判断机构进场
我通常看四个方向：

一，看行业，权重占50%

行业这点非常重要，机构不是游资，高价低价好股渣股全都买，机构能介入的行业往往离不开业绩逻辑驱动，即便他们参与投机也不是单纯的情绪投机。

有一个简单办法，机构们平时推荐的方向最容易有机构席位买入，比如之前的消费电子、半导体，机构天天讲，这类票涨停多数都落不下机构席位，比如水泥板块，机构经常“吹”，昨天天山股份涨停，龙虎榜有机构席位，比如家装板块，今天海鸥住工涨停，龙虎榜有机构席位。

但要说某只低价低市值小盘股炒情绪投机涨停，龙虎榜出现机构的席位可能性几乎是零，比如前几天那些准ST股，ST实达已经三个跌停了，还有之前的暴风集团，几乎不会有机构席位买入，再比如省广大帝，再NB他也不是机构的菜，鲜有机构买。

二，看盘口，权重占20%

机构买入盘口经常有一些规律性，典型的整数位的买单，比如100手，200手等一些规律买单，从金额上看如果规律性单子大概三五十万，有效性就更高了。

很简单道理，如果是游资买一只票，没必要或者犯不上一二百手不断买入，不都想体现下大单点火的快感？通常机构想要拿筹码才会那么干。

三，看分时，权重占20%

机构分时和游资分时图差别很大，机构分时给人感受是不差钱，经常生拉硬拽，分时走势比较陡，而游资分时比较委婉，在大资金不点火时经常横着多一些。

机构席位买入，盘中分时给人感受是连续吃货拿筹码，如上图的海鸥住工，而游资席位买入，分时给人感受是合力居多，一波资金拉完，换一波资金再拉升。

四，其他，占10%

如果是机构买入，多数票在炸板后尾盘经常会做修复性回封，以让个股符合上龙虎榜条件。机构资金不傻，知道席位溢价后喜欢展示给市场，借着锁仓的美名，次日有更多资金甘心去接力。
