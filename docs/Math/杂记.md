## 增根

[由 x²+x+1=0 得到 3=0 错在何处？](https://www.zhihu.com/question/309863493)

$$
\begin{equation} 
\begin{aligned}

x^ 2 + x + 1 = 0 && (x \neq 0 \to x + 1 = -x^2) \\

\therefore x + 1 + \frac{1}{x} = 0 && (x \neq 0) \\

\therefore -x^{2} + \frac{1}{x} = 0 \\

\therefore \frac{1}{x} = x^2 \\

\therefore x^3 = 1 \\

\end{aligned}
\end{equation}
$$

显然增根$x = 1$ 出现在**3式**，原因是由于**变量替换需要全部替换，不能部分替换**。

注意到**2式**减去**1式**也可以得到**3式**。

则由下面做法可以更清楚看出增根的由来。首先记 $A = x^ 2 + x + 1$

$$
\begin{equation} 
\begin{aligned}

A = 0 \\
\frac{A}{x} = 0 && (x \neq 0) \\

A - \frac{A}{x} = 0 \\
(1 -x)A = 0 \\
(1 -x)(x^2 + x + 1) = 0\\
x^3 = 1

\end{aligned}
\end{equation}
$$

## 圆锥曲线

[高中数学解析几何圆锥曲线大题真的是硬解就能解出吗？](https://www.zhihu.com/question/347025945/answer/1186733487)

使用$(x, y)$ 处理圆锥曲线的难点在于，它没有保持结构，而是将其看出孤立的点。 对于二次曲线和直线的交点，韦达定理好用的原因就是它揭示了交点符合的数学结构，对这类问题，一定要跳出去整体去看 $(x, y)$，而不是单独看 $x$ 或者 $y$ 。

> [!warning] 其实就是大学中的线代和矩阵


例子一：

求斜率为定值： $k = \frac{y-n}{x-m}$  。 则用 $y'=y-n$  和 $x'=x-m$ 代入原曲线和直线，保持整体。（其实就是矩阵线性变化）。

例子二：
有时由韦达定理得到的 $x_{1} + x_{2}$ , $x_{1}x_{2}$ 是线性相关的。即  

$$ x_{1}x_{2} = g(m) * (x_{1} + x_{2})$$

这种情况其实只有一个自由度， 需要保持一个自由度去解决非对称韦达的问题。
