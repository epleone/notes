# webdataset

## preface

[webdataset](https://webdataset.github.io/webdataset/)是一个数据加载的库，其可以从tar文件中直接读取数据样本而无需将tar包中的所有文件释放出来。webdataset的主要目的是为了解决传统数据加载方式（就是直接从磁盘中加载大量数据集文件）存在的一些问题。
