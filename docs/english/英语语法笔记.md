## 时态

时态就是时 + 态。时：时间；态：状态

其中时间又分过去、现在、将来。

状态又分一般、进行、完成。

|   时间/状态   |     一般式      |       完成式       |       进行式        |         完成进行式         |
| :-------: | :----------: | :-------------: | :--------------: | :-------------------: |
|  **过去时**  |   did + do   |   had + done    | did + be + doing |      had + doing      |
|  **现在时**  | do/does + do | have/has + done |    be + doing    | have/has + be + doing |
|  **将来时**  |              |                 |                  |                       |
| **过去将来时** |              |                 |                  |                       |

句子 = 主语 +  谓语部分。

谓语部分 = 谓语动词 + 其它

谓语动词 =  助动词 + 实义动词
