# 基础知识
## Byte 和 bit

字节（Byte）是计算机中数据存储和传输的基本单位。一个字节通常由8个二进制位（bit）组成，每个二进制位可以表示 0 或者 1。因此，一个字节可以表示 2^8 = 256 种不同的状态。

一些常见的数据类型及其对应的字节数: 

| 数据类型      | 字节数      |
| --------- | -------- |
| char      | 1 字节     |
| _Bool     | 1 字节     |
| short     | 2 字节     |
| int       | 4 字节     |
| float     | 4 字节     |
| long      | 4 或 8 字节 |
| long long | 8 字节     |
| double    | 8 字节     |
| size_t    | 4 或 8 字节 |
| 指针类型      | 4 或 8 字节 |

在 32 位系统上，指针需要 4 字节来存储地址；
在 64 位系统上，指针需要 8 字节来存储地址。

需要注意的是，这只是一般情况下的规则，具体的指针大小可能会因为编译器、操作系统或者编译选项的不同而有所变化。

您可以使用 `sizeof` 运算符来获取指针的大小，例如 `sizeof(int*)` 可以获取 `int*` 类型的指针在当前平台上的大小。


## 定义和声明

**声明 (Declaration)**:
- 声明是对变量、函数或类的简单介绍，告诉编译器这些符号的名称和类型。声明告诉编译器，这些符号是存在的，但不提供详细信息。

``` c++
// 函数声明
int add(int a, int b);


// 变量声明
extern int globalVar;

```

**定义 (Definition)**:
- 定义提供了有关变量、函数或类的详细信息，包括其内部实现。

``` c++

// 函数定义
void function() 
{ 
	/* 函数体 */ 
}

// 变量定义
int variable = 10;

```


**类和结构体**

``` c++

// 前向声明
class MyClass;


// 类的声明
class MyClass {
public:
    void someMethod();
    int someVariable;
};


// 类的定义
#include "MyClass.h" // 包含类的声明

void MyClass::someMethod() {
    // 方法实现
}

```


- **前向声明**：通常用于解决循环依赖问题、减少编译时间或隐藏实现细节。但编译器不知道这个类的具体成员、大小等信息。
- **声明**：提供了类的完整描述，包括其成员、方法声明等。
- **定义**：提供了类的完整实现，包括其成员函数的具体实现、全局成员变量的定义等。

一般来说，类的成员函数都是inline, 可以直接在声明中提供而不引发编译错误，但如果类的非内联函数或静态成员变量在头文件中定义了，也会引起重定义错误，就像变量和函数一样。
模板类不一样，全部都是都是inline, 在link的时候直接插入到当前位置。
