    git з®ҖжҳҺж•ҷзЁӢ¶

    жҺЁиҚҗйҳ…иҜ»В git - з®Җжҳ“жҢҮеҚ—

    GitжңҖдҪіе®һи·өпјҢиҝҷж ·з”Ёе°ұеҜ№дәҶ

    еҗ„дҪҚеӨ§дҪ¬дҪ д»¬еӣўйҳҹејҖеҸ‘ git жҳҜеҰӮдҪ•з®ЎзҗҶзҡ„пјҹ - V2EX

    вҖӢ вҖӢ жң¬ең°д»“еә“з”ұ git з»ҙжҠӨзҡ„дёүжЈөвҖңж ‘вҖқз»„жҲҗгҖӮ

    • з¬¬дёҖдёӘжҳҜдҪ зҡ„е·ҘдҪңзӣ®еҪ•пјҢе®ғжҢҒжңүе®һйҷ…ж–Үд»¶пјӣ

    • з¬¬дәҢдёӘжҳҜжҡӮеӯҳеҢәпјҲIndexпјүпјҢе®ғеғҸдёӘзј“еӯҳеҢәеҹҹпјҢдёҙж—¶дҝқеӯҳдҪ зҡ„ж”№еҠЁпјӣ

    • жңҖеҗҺжҳҜВ HEADпјҢе®ғжҢҮеҗ‘дҪ жңҖеҗҺдёҖж¬ЎжҸҗдәӨзҡ„з»“жһңгҖӮ

    еҲӣе»әж–°д»“еә“¶

    вҖӢ еҲӣе»әж–°ж–Үд»¶еӨ№пјҢжү“ејҖпјҢз„¶еҗҺжү§иЎҢ

    git init

    clone: е…ӢйҡҶд»“еә“¶

    вҖӢ жү§иЎҢеҰӮдёӢе‘Ҫд»Өд»ҘеҲӣе»әдёҖдёӘжң¬ең°д»“еә“зҡ„е…ӢйҡҶзүҲжң¬пјҡ

    git clone /path/to/repository

    вҖӢ еҰӮжһңжҳҜиҝңз«ҜжңҚеҠЎеҷЁдёҠзҡ„д»“еә“пјҢдҪ зҡ„е‘Ҫд»ӨдјҡжҳҜиҝҷдёӘж ·еӯҗпјҡ

    git clone https://github.com/epleone/epleone.github.io.git

    add: ж·»еҠ е’ҢжҸҗдәӨ¶

    вҖӢ дҪ еҸҜд»ҘжҸҗеҮәжӣҙж”№пјҲжҠҠе®ғд»¬ж·»еҠ еҲ°жҡӮеӯҳеҢәпјүпјҢдҪҝз”ЁеҰӮдёӢе‘Ҫд»Өпјҡ

    git add <filename>
git add *
git add ./

    вҖӢ иҝҷжҳҜ git еҹәжң¬е·ҘдҪңжөҒзЁӢзҡ„з¬¬дёҖжӯҘгҖӮ

    вҖӢ дҪҝз”ЁеҰӮдёӢе‘Ҫд»Өд»Ҙе®һйҷ…жҸҗдәӨж”№еҠЁпјҡ

    git commit -m "д»Јз ҒжҸҗдәӨдҝЎжҒҜ"

    вҖӢ зҺ°еңЁпјҢдҪ зҡ„ж”№еҠЁе·Із»ҸжҸҗдәӨеҲ°дәҶ HEADпјҢдҪҶжҳҜиҝҳжІЎеҲ°дҪ зҡ„иҝңз«Ҝд»“еә“гҖӮ

    push: жҺЁйҖҒж”№еҠЁ¶

    вҖӢ дҪ зҡ„ж”№еҠЁзҺ°еңЁе·Із»ҸеңЁжң¬ең°д»“еә“зҡ„ HEAD дёӯдәҶгҖӮжү§иЎҢеҰӮдёӢе‘Ҫд»Өд»Ҙе°Ҷиҝҷдәӣж”№еҠЁжҸҗдәӨеҲ°иҝңз«Ҝд»“еә“пјҡ

    git push origin master

    вҖӢ еҸҜд»ҘжҠҠ master жҚўжҲҗдҪ жғіиҰҒжҺЁйҖҒзҡ„д»»дҪ•еҲҶж”ҜгҖӮ

    вҖӢ еҰӮжһңдҪ иҝҳжІЎжңүе…ӢйҡҶзҺ°жңүд»“еә“пјҢе№¶ж¬Іе°ҶдҪ зҡ„д»“еә“иҝһжҺҘеҲ°жҹҗдёӘиҝңзЁӢжңҚеҠЎеҷЁпјҢдҪ еҸҜд»ҘдҪҝз”ЁеҰӮдёӢе‘Ҫд»Өж·»еҠ пјҡ

    git remote add origin <server>

    вҖӢ еҰӮжӯӨдҪ е°ұиғҪеӨҹе°ҶдҪ зҡ„ж”№еҠЁжҺЁйҖҒеҲ°жүҖж·»еҠ зҡ„жңҚеҠЎеҷЁдёҠеҺ»дәҶгҖӮ

    checkout + branch: еҲҶж”Ҝз®ЎзҗҶ¶

    gitдёӯзҡ„еҲҶж”Ҝз®ЎзҗҶпјҡgit branch,git checkoutпјҢи§ЈеҶіgitдёӯзҡ„еҲҶж”ҜеҶІзӘҒзҡ„ж–№жі•

    GitеҲҶж”Ҝ(branch)иҜҰз»Ҷи®Іи§Ј_git branch-CSDNеҚҡе®ў

    вҖӢ еҲҶж”ҜжҳҜз”ЁжқҘе°Ҷзү№жҖ§ејҖеҸ‘з»қзјҳејҖжқҘзҡ„гҖӮеңЁдҪ еҲӣе»әд»“еә“зҡ„ж—¶еҖҷпјҢmaster жҳҜвҖңй»ҳи®Өзҡ„вҖқеҲҶж”ҜгҖӮеңЁе…¶д»–еҲҶж”ҜдёҠиҝӣиЎҢејҖеҸ‘пјҢе®ҢжҲҗеҗҺеҶҚе°Ҷе®ғд»¬еҗҲе№¶еҲ°дё»еҲҶж”ҜдёҠгҖӮ

    • жҹҘзңӢеҲҶж”Ҝпјҡgit branch
    • еҲӣе»әеҲҶж”Ҝпјҡgit branch <name>
    • еҲҮжҚўеҲҶж”Ҝпјҡgit checkout <name> жҲ–иҖ… git switch <name>
    • еҲӣе»әе№¶еҲҮжҚўеҲҶж”Ҝпјҡgit checkout -b <name> жҲ–иҖ… git switch -c <name>
    • еҗҲе№¶жҹҗдёӘеҲҶж”Ҝпјҡgit merge <name>
    • еҲ йҷӨеҲҶж”Ҝпјҡgit branch -d <name>

    git checkoutВ е’ҢВ git switchВ жҳҜВ GitВ дёӯз”ЁдәҺеҲҮжҚўеҲҶж”Ҝзҡ„е‘Ҫд»Ө, git switchВ жҳҜеңЁ Git 2.23 зүҲжң¬еј•е…Ҙзҡ„пјҢе®ғжҸҗдҫӣдәҶдёҖз§Қжӣҙзӣҙи§Ӯзҡ„еҲҶж”ҜеҲҮжҚўж–№ејҸгҖӮ

    git checkoutе’Ңgit switchзҡ„еҢәеҲ«_git switchе’Ңgit checkout-CSDNеҚҡе®ў

    дҪҝз”Ёcheckout жқҘеҲӣе»әе’ҢеҲҮжҚўеҲҶж”ҜгҖӮ

    еҲӣе»әеҲҶж”Ҝ¶

    вҖӢ еҲӣе»әдёҖдёӘеҸ«еҒҡвҖңfeature_xвҖқзҡ„еҲҶж”ҜпјҢе№¶еҲҮжҚўиҝҮеҺ»пјҡ

    git checkout -b feature_x

    еҲҮжҚўеҲҶж”Ҝ¶

    вҖӢ еҲҮжҚўеӣһдё»еҲҶж”Ҝпјҡ

    git checkout master

    жҹҘзңӢеҲҶж”Ҝ¶

    git branch

    еҲ йҷӨеҲҶж”Ҝ¶

    вҖӢ еҶҚжҠҠж–°е»әзҡ„еҲҶж”ҜеҲ жҺүпјҡ

    git branch -d feature_x

    вҖӢ йҷӨйқһдҪ е°ҶеҲҶж”ҜжҺЁйҖҒеҲ°иҝңз«Ҝд»“еә“пјҢдёҚз„¶иҜҘеҲҶж”Ҝе°ұжҳҜ дёҚдёәд»–дәәжүҖи§Ғзҡ„пјҡ

    git push origin <branch>

    еҲҶзҰ»еӨҙжҢҮй’ҲпјҲdetached HEADпјү¶

    еҪ“дҪ еҲҮжҚўеҲ°дёҖдёӘеҺҶеҸІжҸҗдәӨж—¶пјҢGit дјҡиҝӣе…ҘвҖңеҲҶзҰ»еӨҙжҢҮй’ҲвҖқзҠ¶жҖҒпјҢжӯӨж—¶дҪ дёҚжҳҜеңЁдёҖдёӘеҲҶж”ҜдёҠе·ҘдҪңпјҢиҖҢжҳҜеңЁдёҖдёӘзү№е®ҡзҡ„жҸҗдәӨдёҠе·ҘдҪңгҖӮеҰӮжһңдҪ еңЁиҝҷз§ҚзҠ¶жҖҒдёӢиҝӣиЎҢдҝ®ж”№е№¶жҸҗдәӨпјҢж–°зҡ„жҸҗдәӨдёҚдјҡеңЁд»»дҪ•еҲҶж”ҜдёҠгҖӮеӣ жӯӨпјҢеҪ“дҪ еҲҮжҚўеӣһе…¶д»–еҲҶж”Ҝж—¶пјҢдҪ дјҡеҸ‘зҺ°еҲҡеҲҡзҡ„жҸҗдәӨвҖңж¶ҲеӨұдәҶвҖқпјҢе®һйҷ…дёҠе®ғд»¬еҸӘжҳҜжІЎжңүйҷ„еҠ еҲ°д»»дҪ•еҲҶж”ҜдёҠгҖӮ

    дҪ еҸҜд»ҘдҪҝз”Ё git reflog жқҘжүҫеҲ°жңҖиҝ‘зҡ„жҸҗдәӨгҖӮgit reflog дјҡи®°еҪ•жүҖжңү HEAD зҡ„еҸҳеҠЁпјҢеҢ…жӢ¬еңЁеҲҶзҰ»еӨҙжҢҮй’ҲзҠ¶жҖҒдёӢзҡ„жҸҗдәӨгҖӮ

    # жүҫеҲ°дҪ жғіиҰҒзҡ„жҸҗдәӨзҡ„е“ҲеёҢеҖј
git reflog

    pull = fetch + merge: жӣҙж–°дёҺеҗҲе№¶¶

    вҖӢ иҰҒжӣҙж–°дҪ зҡ„жң¬ең°д»“еә“иҮіжңҖж–°ж”№еҠЁпјҢжү§иЎҢпјҡ

    git pull

    д»ҘеңЁдҪ зҡ„е·ҘдҪңзӣ®еҪ•дёӯ иҺ·еҸ–пјҲfetchпјү е№¶ еҗҲе№¶пјҲmergeпјү иҝңз«Ҝзҡ„ж”№еҠЁгҖӮ
